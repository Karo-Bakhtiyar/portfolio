# وشەکان

**A Wordle clone built entirely in Sorani Kurdish, because nobody had done it properly yet.**

[Live Demo](https://karo-bakhtiyar.github.io/wshakan-app) · [Repo](https://github.com/Karo-Bakhtiyar/wshakan-app)

</div>

---

Wordle took over the internet in 2022. Nearly every language got a version. Kurdish didn't — at least not one that actually worked: proper Sorani Arabic script, real RTL layout, a keyboard that makes sense, and a word list built from words Kurdish speakers actually use.

وشەکان is that version.

## What it does

Players guess a hidden Sorani Kurdish word in 6 attempts. After each guess, tiles show which letters are correct, present but misplaced, or absent. Word length is adjustable — 4, 5, or 6 letters — with a separate word list for each.

The on-screen keyboard renders in Sorani Arabic script and updates color as the player guesses. Physical keyboard input is also supported via a Latin-to-Kurdish mapping, so players don't need to switch their OS keyboard.


## Stack

Pure HTML, CSS, and JavaScript. No frameworks, no dependencies, no build step. Opens directly in a browser.

یاری Wordle لە ساڵی ٢٠٢٢دا جیهانی داگیر کرد. نزیکەی هەموو زمانەکان وەشانێکی تایبەت بە خۆیان بۆ دروست کرا. کوردی بێبەش بوو — یان لانی کەم وەشانێکی وای نەبوو کە بە دروستی کار بکات: بە ئەلفوبێی کوردیی سۆرانی، ڕووکارێکی ڕاست-بۆ-چەپ (RTL)، کیبۆردێکی گونجاو، و لیستێکی دەوڵەمەند لەو وشانەی کە کوردەکان لە ژیانی ڕۆژانەدا بەکاریان دەهێنن.

وشەکان ئەو وەشانەیە.

## چی دەکات؟

یاریزانەکان هەوڵ دەدەن وشەیەکی شاراوەی کوردیی سۆرانی لە ٦ هەوڵدا بدۆزنەوە. دوای هەر هەوڵێک، ڕەنگی خانەکان دەگۆڕێت بۆ نیشاندانی ئەوەی کام پیت ڕاستە، کام پیت لە وشەکەدا هەیە بەڵام شوێنەکەی هەڵەیە، و کام پیت لە وشەکەدا بوونی نییە. درێژی وشەکان دەکرێت دەستکاری بکرێت — ٤، ٥، یان ٦ پیت — لەگەڵ لیستێکی جیاواز بۆ هەر قەبارەیەک.

کیبۆردی سەر شاشەکە بە ئەلفوبێی کوردیی سۆرانی دیزاین کراوە و لەگەڵ هەر هەوڵێکی یاریزانەکە ڕەنگەکەی نوێ دەبێتەوە. هەروەها پشتگیری کیبۆردی دەرەکی (Physical Keyboard) دەکات لە ڕێگەی نەخشەیەک بۆ گۆڕینی پیتە لاتینییەکان بۆ کوردی، بۆ ئەوەی یاریزانەکە پێویست نەکات زمانی کیبۆردی سیستەمەکەی بگۆڕێت.


## تەکنەلۆژیا (Stack)

تەنها HTML، CSS، و JavaScript. بێ بەکارهێنانی هیچ فریمۆرک، کتێبخانە، یان هەنگاوی بونیادنان (Build step). ڕاستەوخۆ لە وێبگەڕدا دەکرێتەوە.
