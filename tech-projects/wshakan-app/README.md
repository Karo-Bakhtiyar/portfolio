# وشەکان — Kurdish Wordle

A Wordle-style word guessing game built in Sorani Kurdish. Players guess a hidden Kurdish word in 6 attempts, with color-coded feedback after each guess.

Supports 4, 5, and 6 letter words. Full RTL layout, on-screen Sorani Arabic script keyboard, and physical keyboard mapping.

**Live Demo:** https://karo-bakhtiyar.github.io/wshakan-app  
**Repo:** https://github.com/Karo-Bakhtiyar/wshakan-app  
**Stack:** HTML, CSS, JavaScript
