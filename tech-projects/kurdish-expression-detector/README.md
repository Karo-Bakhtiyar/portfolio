# Kurdish Expression Detector

A real-time facial expression detection app built with React and Vite that runs entirely in the browser. No API keys, no server, no data leaves the device.

**[Live Demo](https://karo-bakhtiyar.github.io/kurdish-expression-detector)** | **[Repo](https://github.com/Karo-Bakhtiyar/kurdish-expression-detector)**

---

## What it does

Uses two machine learning models running locally in the browser to track your face and read your expression in real-time. The face mesh draws 478 landmark points across your face while the expression classifier identifies your current mood and displays it in Sorani Kurdish.

Supports 7 expressions: happy (دڵخۆش), sad (خەمگین), angry (توڕە), fearful (ترساو), disgusted (بێزار), surprised (سەرسام), and neutral (ئاسایی).

---

## Tech

- React + Vite
- MediaPipe FaceMesh for 478-point real-time face landmark detection
- face-api.js for expression classification
- All model weights bundled locally, fully offline after install

## چی دەکات؟

دوو مۆدێلی (Machine Learning) بەکاردێنێت کە بە شێوەیەکی ناوخۆیی لەناو وێبگەڕەکەدا کاردەکەن بۆ چاودێریکردنی ڕوخسار و خوێندنەوەی دەربڕینەکانت لە کاتی ڕاستەقینەدا. مۆدێلی Face Mesh ٤٧٨ خاڵی نیشانە لەسەرانسەری ڕوخسارتدا دەکێشێت، لەکاتێکدا پۆلێنکەری دەربڕینەکان جۆری مەزاجت دیاری دەکات و بە زمانی کوردی سۆرانی نیشانی دەدات.

پشتگیری لە ٧ دەربڕین دەکات: دڵخۆش، خەمگین، توڕە، ترساو، بێزار، سەرسام، و ئاسایی.
