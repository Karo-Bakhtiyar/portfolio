# Kurdish Prayer Times

A clean, minimal prayer times app built for Kurdish speakers. Shows all 5 daily prayer times with a live countdown to the next prayer, supporting major Kurdish cities and rendered fully in Sorani and badini Kurdish.

**[Live Demo](https://karo-bakhtiyar.github.io/kurdish-prayer-times-app)**

---

## Features

- Prayer times for Sulaymaniyah, Erbil, Duhok, and Kirkuk
- Live countdown timer to the next prayer
- Current date in both Gregorian and Hijri
- Full Sorani & Badini Kurdish UI with RTL layout
- Rabar font for authentic Kurdish text rendering
- Powered by the Aladhan API

---

## How to Run

Download or clone the repo and open `index.html` in any browser. No build process, no dependencies, no internet connection required for the font.

---

## Tech

Pure HTML, CSS, and JavaScript. Single file. Uses the [Aladhan API](https://aladhan.com/prayer-times-api) for prayer time calculation.

# کاتەکانی نوێژ (Kurdish Prayer Times)

ئەپێکی سادە و مۆدێرن بۆ پیشاندانی کاتەکانی بانگ کە تایبەت بۆ کوردزمانان دروست کراوە. کاتەکانی هەر پێنج فەرزەکە پیشان دەدات لەگەڵ ژمێریاریی پێچەوانە (Countdown) بۆ بانگی داهاتوو. پشتگیری شارە سەرەکییەکانی کوردستان دەکات و بە هەردوو شێوەزاری سۆرانی و بادینی بەردەستە.

**[نمونەی زیندوو (Live Demo)](https://karo-bakhtiyar.github.io/kurdish-prayer-times-app)**

---

## تایبەتمەندییەکان

- کاتەکانی بانگ بۆ شارەکانی سلێمانی، هەولێر، دهۆک و کەرکووک.
- ژمێریاریی ڕاستەوخۆ بۆ کاتی ماوە بۆ بانگی داهاتوو.
- پیشاندانی بەروار بە هەردوو ساڵنامەی زاینی و کۆچی.
- ڕووکارێکی تەواو کوردی (سۆرانی و بادینی) بە ئاڕاستەی ڕاست بۆ چەپ (RTL).
- بەکارهێنانی فۆنتی "ڕابەر" بۆ نیشاندانی دەقە کوردییەکان بە شێوازێکی ڕەسەن.
- کاردەکات لە ڕێگەی Aladhan API.

---

## چۆنێتی کارپێکردن

پڕۆژەکە دابەزێنە یان کلۆنی بکە و فایلی `index.html` لە هەر وێبگەڕێکدا بکەرەوە. پێویستی بە هیچ پرۆسەیەکی بونیادنان (Build) یان کتێبخانەیەکی دەرەکی نییە، تەنانەت بۆ فۆنتەکەش پێویستی بە ئینتەرنێت نییە.

---

## تەکنەلۆژیا بەکارهاتووەکان

تەنها HTML، CSS و JavaScript. هەمووی لە یەک فایلدایە. [Aladhan API](https://aladhan.com/prayer-times-api) بەکارهاتووە بۆ هەژمارکردنی کاتەکانی بانگ.
